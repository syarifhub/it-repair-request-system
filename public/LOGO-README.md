# วิธีเพิ่มโลโก้โรงแรม

## ขั้นตอนการเพิ่มโลโก้:

1. **เตรียมไฟล์โลโก้**:
   - ชื่อไฟล์: `logo.png` (หรือ `logo.jpg`, `logo.svg`)
   - ขนาดแนะนำ: 200x200 pixels หรือ 400x400 pixels
   - พื้นหลังโปร่งใส (PNG) จะดูดีที่สุด

2. **วางไฟล์โลโก้**:
   - คัดลอกไฟล์โลโก้มาวางในโฟลเดอร์: `frontend/src/assets/`
   - ตั้งชื่อไฟล์เป็น: `logo.png`
   - **สำคัญ**: ต้องวางที่ `frontend/src/assets/logo.png` ไม่ใช่ `frontend/public/`

3. **ตำแหน่งที่โลโก้จะแสดง**:
   - ✅ หน้าแรก (Home Page) - ด้านบนสุด
   - ✅ หน้า Login Admin - ด้านบนฟอร์ม
   - ✅ Tab icon (Favicon) ในเบราว์เซอร์

4. **ทดสอบ**:
   - รีเฟรชหน้าเว็บ (Ctrl + Shift + R)
   - ถ้าไม่เห็นโลโก้ ให้ตรวจสอบว่าชื่อไฟล์ถูกต้อง

## ตัวอย่างโครงสร้างไฟล์:

```
frontend/
├── src/
│   ├── assets/
│   │   ├── logo.png      ← วางไฟล์โลโก้ที่นี่ (สำคัญ!)
│   │   └── react.svg
│   └── pages/
├── public/
│   ├── vite.svg
│   └── LOGO-README.md    ← ไฟล์นี้
```

## หมายเหตุ:

- ถ้าไม่มีโลโก้ ระบบจะแสดง emoji 🏨 หรือ 🔐 แทน
- รองรับไฟล์: PNG, JPG, SVG
- ขนาดไฟล์ไม่ควรเกิน 500KB เพื่อความเร็วในการโหลด
- **ปัญหาที่พบบ่อย**: ถ้าโลโก้เป็นสีขาวหรือโปร่งใส อาจมองไม่เห็นบนพื้นหลังสีเทา
- **แนะนำ**: ใช้โลโก้ที่มีสีชัดเจน (เช่น สีน้ำเงิน, สีเขียว) จะเห็นได้ชัดเจนกว่า
- หลังจากเปลี่ยนโลโก้ ต้องรีสตาร์ท dev server (ปิด npm run dev แล้วเปิดใหม่)
